lettcode-有效括号

Posted on 2019-12-15 Edited on 2019-12-28 In 数据结构 Views: Valine:

题目

给定一个只包括 ‘(‘，’)’，’{‘，’}’，’[‘，’]’ 的字符串，判断字符串是否有效。
有效字符串需满足：
左括号必须用相同类型的右括号闭合。
左括号必须以正确的顺序闭合。
注意空字符串可被认为是有效字符串

/**
 * 通过堆的方法 进行匹配  先进后出
 */
public static boolean isValid(String s) {
    if (s.length() % 2 != 0) return false;
    if (s.length() == 0 || "".equals(s)) return true;

    char[] chars = s.toCharArray();
    Stack<Character> stack = new Stack<>();
    for (int i = 0; i < chars.length; i++) {
        if (stack.size() >= 1 
            && getOpposite(stack.peek()) == chars[i]) {
            stack.pop();
            continue;
        }
        stack.push(chars[i]);
    }
    if(stack.size() == 0) return true;

    return false;
}

private static char getOpposite(char c) {
    switch (c) {
        case '(':
            return ')';
        case '{':
            return '}';
        case '[':
            return ']';
    }
    return ' ';
}
